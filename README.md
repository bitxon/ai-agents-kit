# AI Agents Kit

A catalog of skills, agents, and related AI building blocks.

## Skills

- **[jvm-ktor-scaffold](skills/jvm-ktor-scaffold/SKILL.md)** — Scaffolds a new Kotlin Ktor microservice or REST API project using start.ktor.io
- **[jvm-quarkus-scaffold](skills/jvm-quarkus-scaffold/SKILL.md)** — Scaffolds a new Java or Kotlin Quarkus microservice or REST API project using code.quarkus.io
- **[jvm-spring-scaffold](skills/jvm-spring-scaffold/SKILL.md)** — Scaffolds a new Kotlin or Java Spring Boot microservice or REST API project using Spring Initializr
- **[jvm-build-doctor](skills/jvm-build-doctor/SKILL.md)** — Diagnoses and resolves Maven and Gradle build failures: dependency resolution, version conflicts, compilation errors, plugin failures, and JVM/toolchain mismatches
- **[jvm-spring-migrate-localstack-to-floci](skills/jvm-spring-migrate-localstack-to-floci/SKILL.md)** — Migrates a Spring Boot project from LocalStack to Floci AWS emulator
- **[rest-api-design](skills/rest-api-design/SKILL.md)** — REST API design standards: endpoint naming, HTTP methods, status codes, pagination, error responses, versioning
